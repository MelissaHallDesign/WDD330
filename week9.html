Notes and Code Examples from JavaScript: Novice to Ninja, 2nd Edition

The Browser Object Model (or BOM for short) is a collection of properties and methods that contain information about the browser and computer screen.

The Window Object

In a browser environment the global object is the window object, which represents the browser window that contains a web page.

The Document Object

Each window object contains a document object. (document.write, etc)

Cookies

Cookies are also limited to storing up to 4KB of data, although 20 cookies are allowed per domain, which can add up to quite a lot of data.

Cookies take the form of a text file that contain a list of name/value pairs separated by semicolons. For example, a cookie file might contain the following information:

"name=Superman; hero=true; city=Metropolis"

Creating cookies

To create a cookie, you assign it to JavaScript’s “cookie jar”, using the document.cookie property, like so:

document.cookie = 'name=Superman';
"name=Superman"

APIs

The Geolocation API is used to obtain the geographical position of the device.
