Notes and Code Examples from HTML5 & CSS3 for the Real World: 2nd Edition

Transforms

The CSS3 transform property lets you lets you translate, rotate, scale, and/or skew any element on the page.

We can manipulate an element’s appearance usingtransform functions

Translation

Translation functions allow you to move elements left, right, up, or down.

The translate(x,y) function moves an element x from the left, and y from the top:

transform: translate(45px, -45px);

Scaling

The scale(x,y) function scales an element by the defined factors horizontally then vertically.

transform: scale(1.5, 0.25);

Rotation

The rotate() function rotates an element around the point of origin by a specified angle value.

.ad-ad2 h1:hover span {
color: #484848;
transform: rotate(10deg) translateX(40px) scale(1.5);
}

Skew

The skew(x,y) function specifies a skew along the x and y axes.

transform: skew(15deg, 4deg);

Canvas

With HTML5’s Canvas API, we can draw anything we can imagine, all through JavaScript.

Canvas is a HTML element


Sorry! Your browser doesn’t support Canvas.

 .myCanvas {
border: dotted 2px black;
}

Coloring our canvas:

var canvas = document.getElementById("myCanvas");
var context = canvas.getContext("2d");
context.strokeStyle = "red";
context.fillStyle = "blue";

Changing the opacity:

var canvas = document.getElementById("myCanvas");
var context = canvas.getContext("2d");
context.strokeStyle = "red";
context.fillStyle = "rgba(0, 0, 255, 0.5)";

XML

XML stands for eXtensible Markup Language. Like HTML, it’s a markup metalanguage. In plain English, it’s a system meant to annotate text. Just as we can use HTML tags to wrap our content and give it meaning, so can XML tags be used to describe data, such as the content of a file.
