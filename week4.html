Notes and Code Examples from JavaScript: Novice to Ninja, 2nd Edition

Object oriented programming in JavaScript

There are three main concepts in OOP: encapsulation, polymorphism and inheritance.

Example of encapsulation:

The inner workings are kept hidden inside the object and only the essential functionalities are exposed to the end user

Example of polymorphism:

The same process can be used for different objects

Example of inheritance:

Take an object that already exists and inherit all its properties and methods.

Java and Ruby, are known as class-based languages

JavaScript is a prototype-based language

Modular JavaScript is a self-contained piece of code that provides functions and methods that can then be used in other files and by other modules.

Code Examples

Example of object literal notation:

const dice = {
sides: 6,
roll() {
return Math.floor(this.sides * Math.random() + 1)
}
}

Example of same code using constructor function:

const Dice = function(sides=6){
this.sides = sides;
this.roll = function() {
return Math.floor(this.sides * Math.random() + 1)
}
}
