Week 3

Notes and Code Examples from Object methods, "this"

Example of Object:

let user = {
name: "John",
age: 30
};

Example of Method Shorthand:

// these objects do the same
user = {
sayHi: function() {
alert("Hello");
}
};
// method shorthand looks better, right?
user = {
sayHi() { // same as "sayHi: function()"
alert("Hello");
}
};

A method can use the "this" keyword to access an object.

let user = {
name: "John",
age: 30,

sayHi() {
// "this" is the "current object"
alert(this.name);
}
};
user.sayHi(); // John

Arrow functions don't use the "this" keyword

let user = {
firstName: "Ilya",
sayHi() {
let arrow = () => alert(this.firstName);
arrow();
}
};

user.sayHi(); // Ilya
